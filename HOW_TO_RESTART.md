# 🚀 Project ko Dobara Kaise Start Karein? (How to Restart)

Agar aapne apna computer ya terminal band kar diya hai, toh project ko wapas chalu karne ke liye ye steps follow karein:

### 1. Folder Open Karein
Sabse pehle apne project folder (`Antigratvity new dl model`) mein jayein.

### 2. Terminal ya Command Prompt Open Karein
Folder ke andar right-click karke "Open in Terminal" ya manually `cmd` open karein.

### 3. Server Start Karein
Terminal mein niche di gayi command likhein aur Enter dabayein:
```bash
python app.py
```
Isse aapka Flask server start ho jayega.

### 4. Browser mein Open Karein
Jab terminal mein message aaye "Running on http://127.0.0.1:5000", toh apne browser (Chrome/Edge) mein ye link open karein:
👉 **http://localhost:5000**

---

### 💡 Kuch Zaroori Baatein:
- **Model Training**: Aapko `python train_model.py` roz-raz run karne ki zaroorat **nahi** hai. Wo hum already kar chuke hain aur uska data `models/` folder mein save hai.
- **Dependencies**: Agar aap kisi naye computer par ye chala rahe hain, toh pehle `pip install flask scikit-learn pandas numpy` run karna hoga.
- **Error Aaye Toh?**: Make sure aap sahi folder mein hain jahan `app.py` file rakhi hai.
